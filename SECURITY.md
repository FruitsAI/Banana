# Security Policy

## Reporting a vulnerability
Please do not open a public GitHub issue for security-sensitive reports.

If GitHub Security Advisories are enabled for this repository, use that channel.
Otherwise, contact the repository owner privately through GitHub before sharing
technical details publicly.

## Scope
This repository currently ships unsigned desktop builds for GitHub-hosted
distribution only. Signing, notarization, and auto-update infrastructure are
not part of the current threat model or release pipeline.
